```python
import sys
N = int(sys.stdin.readline())
dp = [-1] * 1001
# 마지막 돌을 가져가면 지는 게임
# 1개, 3개, 4개 의 선택지가 존재
dp[1] = 0 # 창영이 승
dp[2] = 1 # 상근이 승
dp[3] = 0 # 창영이 승
dp[4] = 1 # 상근이 승
dp[5] = 1 # 상근이 승
# 잘 생각해보자.
# 1개가 남았으면 상근이가 진다.(무조건 1개는 가져가야 한다)
# 2개가 남았으면 상근이가 이긴다.(1개를 가져가서 1개 남기면 창영이는 무조건 지게된다.)
# 3개가 남았으면 상근이가 진다.(상근이가 3개를 가져가는건 자살 행위, 1개를 가져간다고 치면 2개가 남는데 그러면 자연스레 창영이도 1개를 가져가서 패)
# 4개가 남았으면 상근이가 이긴다.(3개를 가져가 1개 남겨서 창영이가 지게 만들 수 있다.)
# 5개가 남았으면 상근이가 상근이가 이긴다.(4개를 가져가서 1개를 남기면 이긴다.)
# 6개가 남았다면...? 4개를 가져가면 상근이가 무조건 지게된다. 그 다음 창영이가 1개를 가져가면 그만이다.
# 1개를 가져간다면 창영이가 4개를 가져가서 지게 만들 것이다.
# 3개를 가져간다면...? 창영이가 갑자기 남은 3개를 가져가서 지진 않을 것이고 1개를 가져가게 되는데, 이러면 상근이도 1개를 가져가서 창영이를 지게만들 수 있다.
# 따라서 dp[i-4], dp[i-3], dp[i-1] 중 0인 경우가 단 하나라도 있다면(해당 턴에 있는 사람이 지는 경우) 상근이가 이긴다.

for i in range(5, N+1):
    if dp[i-1] == 0 or dp[i-3] == 0 or dp[i-4] == 0:
        dp[i] = 1
    else:
        dp[i] = 0

if dp[N] == 0:
    print("CY")
else:
    print("SK")


```
